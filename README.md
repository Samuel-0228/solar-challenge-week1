# Solar Data Discovery Challenge

## Setup
1. git clone https://github.com/Samuel-0228/solar-challenge-week1.git
2. python -m venv .venv && source .venv/bin/activate  # Adapt for Windows
3. pip install -r requirements.txt
4. For notebooks: jupyter notebook  # Or streamlit run app/main.py
